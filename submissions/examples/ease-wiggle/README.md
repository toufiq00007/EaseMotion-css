# Wiggle Animation

## What does this do?

Creates a quick, playful side-to-side wiggle animation that runs once and stops.

## How is it used?

```html
<div class="wiggle-element">
  Content
</div>
```

## Why is it useful?

A wiggle animation can draw attention to call-to-action elements, empty states, badges, and interactive UI elements without feeling like an error state. It provides a playful motion effect that aligns with EaseMotion CSS's animation-first philosophy.
