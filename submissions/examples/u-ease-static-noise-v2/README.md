# ease-static-noise

A TV static noise overlay animation inspired by retro CRT displays and glitch effects.

## Features

- Static noise overlay
- Screen flicker effect
- RGB split animation
- CRT scanlines
- Signal monitor UI
- Jitter distortion
- Responsive layout

## Files

- demo.html
- style.css
- README.md

## Animation Effects

### Static Noise
Rapid opacity fluctuations simulate television interference.

### Jitter
Subtle horizontal and vertical movement adds realism.

### RGB Split
Creates a retro display distortion effect.

### Scanlines
Classic CRT monitor appearance.

## Browser Support

- Chrome
- Edge
- Firefox
- Safari

## Usage

Open `demo.html` in a browser.

## Customization

Adjust animation speed:

```css
animation: staticNoise 0.1s infinite;
