ease-input [type="file"]

Styled file inputs using ::file-selector-button with consistent sizing, hover states, and themeable button variants — matching the existing text input styling.

Usage

<!-- Default file input -->
<input class="ease-input" type="file" />

<!-- With primary button style -->
<input class="ease-input is-primary" type="file" />

<!-- Small variant -->
<input class="ease-input ease-input-sm" type="file" />
