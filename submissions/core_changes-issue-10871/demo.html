ease-connection-status

CSS-only connection status indicator with slide-in animation, pulse effect, and minimal JS for online/offline detection.

This demo simulates connection state changes. In a real app, the status bar would be controlled by window.addEventListener('online', ...) and window.addEventListener('offline', ...). The bar slides down when the connection changes and shows a pulse animation while offline.

Usage

<div id="status" class="ease-connection-status">
  <span class="ease-connection-status-icon"></span>
  <span id="text">Online</span>
</div>

<script>
window.addEventListener('offline', function() {
  var el = document.getElementById('status');
  el.className = 'ease-connection-status is-offline';
  document.getElementById('text').textContent = 'No internet connection';
});

window.addEventListener('online', function() {
  var el = document.getElementById('status');
  el.className = 'ease-connection-status is-online';
  document.getElementById('text').textContent = 'You are back online';
});
</script>
