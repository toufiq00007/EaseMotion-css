ease-skeleton

CSS-only skeleton loading components with shimmer animation, dark mode support, and reduced-motion respect.

Skeleton Types

avatar
image
card
text (3 lines)

Card Grid

Usage

<!-- Avatar -->
<div class="ease-skeleton ease-skeleton-avatar"></div>

<!-- Image -->
<div class="ease-skeleton ease-skeleton-image"></div>

<!-- Text lines -->
<div class="ease-skeleton ease-skeleton-text"></div>
<div class="ease-skeleton ease-skeleton-text"></div>

<!-- Card -->
<div class="ease-skeleton ease-skeleton-card"></div>
