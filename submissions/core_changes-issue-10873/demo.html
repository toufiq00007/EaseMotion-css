ease-tabs

CSS-only animated tabs with content fade-transition — no JavaScript required.

Overview

EaseMotion CSS is a lightweight, animation-first CSS framework designed for modern web interfaces. It provides a curated set of reusable components and utility classes that make it easy to build polished, accessible user interfaces without writing custom CSS.

Whether you are building a landing page, a dashboard, or a full-featured web app, EaseMotion CSS gives you the building blocks you need to create beautiful, responsive designs with minimal effort.

Features

EaseMotion CSS includes a rich set of features out of the box:

  • 60+ utility classes for spacing, typography, and layout
  • 20+ animation and motion utility classes
  • Fully customizable design tokens via CSS custom properties
  • Accessible components with ARIA attributes built in
  • Zero runtime JavaScript — pure CSS
  • Dark mode support with automatic detection

Usage

<div class="ease-tabs">

  <input type="radio" name="tab" id="tab-1" class="ease-tab-input" checked />
  <input type="radio" name="tab" id="tab-2" class="ease-tab-input" />
  <input type="radio" name="tab" id="tab-3" class="ease-tab-input" />

  <div class="ease-tab-list">
    <label class="ease-tab-label" for="tab-1">Tab 1</label>
    <label class="ease-tab-label" for="tab-2">Tab 2</label>
    <label class="ease-tab-label" for="tab-3">Tab 3</label>
  </div>

  <div class="ease-tab-panels">
    <div class="ease-tab-panel" data-tab="1">Content 1</div>
    <div class="ease-tab-panel" data-tab="2">Content 2</div>
    <div class="ease-tab-panel" data-tab="3">Content 3</div>
  </div>

</div>
