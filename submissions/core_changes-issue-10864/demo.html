ease-password-strength

CSS-only password strength indicator with dynamic color transitions. Minimal JavaScript handles the password analysis (length, character diversity).

No password Enter at least 8 characters
8+ characters Lowercase Uppercase Digit Special char

Usage

<div class="ease-password-strength">
  <div class="ease-password-strength-bar" id="bar"></div>
  <div class="ease-password-strength-label">
    <span class="ease-password-strength-text" id="text">No password</span>
  </div>
</div>

<!-- JS toggles .is-weak / .is-medium / .is-strong -->
bar.className = 'ease-password-strength-bar';
bar.classList.add('is-strong');
